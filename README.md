# Joke Teller

```
Key words: Gradle, JAR, AAR, Google Cloud Endpoints, Multiple flavors
```

*	Developed a jokeTeller to pull jokes from cloud backend
*	Created a Java library to provide jokes randomly and deployed it to Google App Engine
*	Created an Android library containing an Activity that will display joke
*	Added free and paid flavors to app and implemented AdMob to display ads on free version
* Customized the behavior of the Gradle build tool to manage multi modules
